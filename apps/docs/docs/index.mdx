# LuckyPlans Documentation

LuckyPlans is infrastructure for perpetual DEX analytics, copy-trading research, and deterministic backtesting.

It helps users analyze trader histories, generate simulation plans, compare follower outcomes, and review risk before execution.

This site is written from the real product implementation currently found in:

- `takeshi-su57/lucky-plan-fe`
- `takeshi-su57/lucky-plan-be`
- the local mirror at `../alpha/fe` and `../alpha/be`

The `beta` monorepo hosts this Docusaurus portal and the newer infrastructure work, while the current product behavior still lives in the split frontend and backend repositories above.
