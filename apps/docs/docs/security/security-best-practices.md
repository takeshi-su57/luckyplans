# Security Best Practices

- Only trust official links.
- Never share seed phrases.
- Be careful with fake support accounts.
- Verify domains before connecting wallets or entering credentials.
- Understand that simulation is not financial advice.
- Treat historical leaderboard quality as research input, not a live-safety guarantee.
- Confirm GitHub repo ownership before cloning tooling or adapters.
