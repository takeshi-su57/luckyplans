# Impersonation Warning

Common attack patterns include:

- fake Discord admins
- fake support accounts
- fake airdrop links
- fake GitHub repositories
- fake wallet connection pages

If a message asks for:

- your seed phrase
- a private key
- direct transfer of funds
- an urgent wallet reconnection through an unknown domain

assume it is malicious until proven otherwise through official channels.
