# Roadmap

The roadmap below is stage-based on purpose. It reflects direction without pretending exact dates are already locked.

## Stage 1: Public Foundation

- stable landing and docs experience
- clearer public explanation of simulation, analytics, and research boundaries
- trustworthy official-link and security messaging

## Stage 2: Developer Readiness

- stronger quickstart and env documentation
- better extension guides for platforms, chains, and contract versions
- clearer GraphQL and operational reference pages

## Stage 3: Product Deep-Dives

- deeper leaderboard methodology docs
- more complete simulation examples
- richer operational guidance for data quality and reindexing

## Stage 4: Ecosystem Expansion

- additional platform and chain coverage
- more formal contributor workflows
- stronger public API and architecture narratives as product surfaces stabilize
